About Us Responsive Master

Responsive Master is a web based emulator which displays the behavior and layout of a website in different devices i.e. mobile, tablet, laptop and desktop in the browser.

This tool is built to to show you how visual elements of your site respond to different screen sizes and enables user to view website UI in various devices on a single page.

This tool helps website makers quickly get an indication of how their responsive site will look on the most popular devices.

I have added 17px to the width of the device to cater for the scroll bar.

Steps on how to use Responsive Master?

  1. Enter URL of the website in the test box at the top
  2. Click on Go!
  3. Website will be loaded in a device (iPhone) which is the default device
  4. Change the device from the dropdowns Phone, Tablet, Laptop, Desttop
  5. To change the Device Orientation ( Portrait/ Landscape view ) click on Rotate button

If a blank page is displayed in the device it means the website has a restriction by setting X-Frame-Options to SAMEORIGIN or DENY, You can download a Google chrome extension Ignore X Frame headers which will bypass the x-frame-options header and allow any website to work in Responsive Master.
